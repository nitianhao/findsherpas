How We Did It

The Prism Audit Pipeline

This audit was conducted using our Prism pipeline — a seven-phase automated search quality assessment. 38 test queries were generated across 15 query categories, grouped into 6 core capabilities.

  1. 1

    Site Discovery

    Automated analysis of your site’s structure, navigation categories, brands, and featured products to understand what your search engine should be able to find.

  2. 2

    Test Design

    Site-type-aware selection from a library of 26 test categories. Categories are chosen based on your site’s commerce model to ensure the audit tests what matters most for your customers.

  3. 3

    Query Generation

    AI-generated realistic customer queries grounded in the actual products, brands, and categories found on your site — not generic templates.

  4. 4

    Result Collection

    Automated search execution collecting the top results for each test query exactly as your customers would see them.

  5. 5

    Relevance Scoring

    Each result scored on a 0.0–1.0 scale by a specialized relevance model, producing an objective measure of how well each result matches the query intent.

  6. 6

    Failure Analysis

    Each query classified by failure mode (e.g., poor ranking, constraint dropped, no fuzzy matching) and severity based on how far the best result was buried below irrelevant ones.

  7. 7

    Report Assembly

    Findings synthesized into this actionable narrative with prioritized recommendations ordered by business impact.
